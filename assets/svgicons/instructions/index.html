Setting Up Your HTML

  1. Open the HTML file where you'd like to put the Animated Icons.
  2. Add the following code inside the head tags.
    3. <script src="http://cdnjs.cloudflare.com/ajax/libs/gsap/latest/TweenMax.min.js"/></script>
<script src="js/noframework.waypoints.min.js"></script>
<script src="js/dxcicons.js" type="text/javascript"/></script>
<script src="js/dxcweathericons.js" type="text/javascript"/></script>
<script src="js/snap.svg.js" type="text/javascript"></script>
<script src="http://code.jquery.com/jquery-latest.min.js" type="text/javascript"></script>
<link rel="stylesheet" type="text/css" href="css/dxcweathericons.css">
  3. You'll need to add a div for each icon. Just copy and paste the divs below based on which one(s) you want to use.
    4. <div class="weathericon clearNightIcon"></div>
<div class="weathericon lightningIcon"></div>
<div class="weathericon rainyIcon"></div
  4. If you'd like to add button to control the icons, use the following code. The numbers in between the [] are the icons you want to affect. For example, if you wanted to do somthing to the first, third, and fourth (remember, counting starts with 0) icons, you would add the following code: 
    5. <button onclick="restartIcons([0,2,3])">Restart</button>

Setting Up Your Javascript File (Optional)

  1. Open js/dxcicons.js
  2. The following can be easily modified in lines 2 - 7
    1. Change the path to the images
    2. Modify the stagger between the animations of each icon
    3. Determine whether or not to have a rollover animation
    4. Determine how much bigger the icon is on rollower (1 = stays the same)
    5. Determine speed of the animation (1 = normal speed)
    6. Determine whether icons should start animating right when the page loads
  3. On line 10 you can add link URL's to each icon. The first spot in the array coorisponds to the first icon. If one of the icons won't have a link, leave a blank spot (,,) In this example, only the first and fourth icons have a link. 
    4. var iconLinks = ["http://aprendagames.com",,,"http://aprendagames.com"];

Upload Your Files

  1. Log into your hosting space via FTP software (I recommend Transmit).
  2. Upload the files to your server.
